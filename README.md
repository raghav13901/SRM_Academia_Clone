# SRM Academia Clone

## Quick Start

- Fork and Clone the repo using
```
git clone https://github.com/raghav13901/SupDB.git
cd SupDB
```
- Install dependencies using
```
npm install
```
- Run server using
```
npm run dev
```
- Then in Browser

localhost:3000
